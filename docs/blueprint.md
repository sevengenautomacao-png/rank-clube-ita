# **App Name**: Pathfinder Ranker

## Core Features:

- Unit Dashboard: Display the different Pathfinder units (Monte Hope, Monte Sinai, Rubi, Sinai) as individual cards.
- Member Management: Allow adding new members to a unit with the specified parameters (name, age, function/cargo, class).
- Member Deletion: Allow removing members from a unit.
- Responsive Design: Ensure the application is optimized for smartphone use, providing a seamless user experience on mobile devices.
- Unit Ranking: Display the members within each unit and a tool that allow admins to update the member rankings based on customizable criteria.

## Style Guidelines:

- Primary color: Yellow (#FFEA00) to evoke a sense of energy and vibrancy.
- Background color: Light gray (#333333) to provide a dark theme and make the yellow elements stand out.
- Accent color: Light yellow (#FFFACD) as an analogous shade with a bit more lightness to call attention to important elements and actions.
- Body and headline font: 'PT Sans' sans-serif font for clear readability on mobile devices.
- Card-based layout for units and members for easy navigation and organization on mobile screens.
- Use simple, clear icons to represent unit functions and member actions.
